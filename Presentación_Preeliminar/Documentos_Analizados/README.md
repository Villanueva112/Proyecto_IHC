# Documentos Analizados – Presentación Preliminar

En esta carpeta se encuentran los **documentos que fueron analizados como parte de la investigación previa para la presentación preliminar del proyecto**.  
Estos materiales sirvieron como base para comprender el contexto del problema, revisar soluciones existentes y obtener información relevante sobre la **Lengua de Señas Mexicana (LSM)**, tecnologías relacionadas y datos estadísticos.
