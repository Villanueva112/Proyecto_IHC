# Presentación Preliminar – Proyecto IHC  
## MY TALKING HANDS

Esta carpeta contiene la **presentación preliminar del proyecto de Interacción Humano-Computadora (IHC)** titulada **"My Talking Hands"**.

El objetivo de esta presentación es **exponer el problema social identificado, la justificación del proyecto, el análisis de soluciones existentes y las bases conceptuales para el diseño de la aplicación**.
